\section{カシミール効果（1948年）}

\begin{storybox}[何もない空間に力がある]
2枚の金属板をナノメートルの距離で向かい合わせると、板の間に引力が生じる。電荷もなく、磁石でもないのに。

原因: 量子力学では「完全に何もない」ことが不可能。真空は常に微小な電磁場の揺らぎ（ゼロ点揺動）で満たされている。板を置くと揺らぎのパターンが変わり、板の間のエネルギーが外より低くなる。エネルギーが低い方に引かれる → 引力。

1948年にカシミールが理論的に予測。1997年に精密測定で確認。
\end{storybox}

\section{揺らぎの「倍音」}

\begin{mathbox}[モードと倍音]
板の間の電磁場には、ギターの弦と同じように倍音（モード）がある:
$$f_1, \; 2f_1, \; 3f_1, \; 4f_1, \; 5f_1, \ldots$$

各モードがゼロ点エネルギー $\hbar\omega/2$ を持つ。全モードの合計が真空エネルギー。

合計: $1 + 2 + 3 + 4 + 5 + \cdots$（1Dの場合）

1D（パイプ）なら $\zeta(-1) = 1+2+3+\cdots = -1/12$。

しかし実際の実験は3D（平行板）。3Dでは横方向の自由度が加わり、和は $\sum n^3$ になる:
$$\zeta(-3) = 1^3 + 2^3 + 3^3 + 4^3 + \cdots = +\frac{1}{120}$$

カシミールエネルギー密度: $\rho = -\pi^2\hbar c/(720 a^4)$。
マイナス符号は3Dの積分から来る。$720 = 6/\zeta(-3)$。

結果: 板の間のエネルギーが外より\textbf{低い} → \textbf{引力}。

これがカシミール効果の数学的正体。実験で精度1\%以内で確認済み。
\end{mathbox}

\section{Boyerの予測（1974年）}

\begin{resultbox}[もし片端を変えたら？]
Boyer は考えた: 片方の板を「完全磁気導体（PMC）」に変えたらどうなるか？

通常の板（PEC）: 電場がゼロになる（ディリクレ境界条件）
PMC: 磁場がゼロになる（ノイマン境界条件）

PEC-PMC にすると、残るモードは奇数だけ:
$$1, \; 3, \; 5, \; 7, \; 9, \ldots$$

3Dでの合計（$\sum n^3$ の奇数版）:
$$\zeta_{\neg 2}(-3) = 1^3 + 3^3 + 5^3 + 7^3 + \cdots = -\frac{7}{120}$$

3Dカシミールエネルギー密度は $\rho \propto -\zeta(-3)$ なので:
$$\rho_{DN} \propto -\zeta_{\neg 2}(-3) = -(-7/120) = +\frac{7}{120} > 0$$

\textbf{正！ 符号が反転した！ しかも大きさは通常の7倍！}

板の間のエネルギーが外より高い → \textbf{斥力}。板が反発する。

（1Dでも同様: $\zeta(-1) = -1/12 \to \zeta_{\neg 2}(-1) = +1/12$ で反転。）
\end{resultbox}

\begin{honestbox}[50年間誰もこれを実験で確認していない]
理由: PMC（完全磁気導体）が存在しない。

普通の金属は「完全電気導体」に近い。しかし「完全磁気導体」として振る舞う物質は、自然界にもメタマテリアルでも実現されていない（カシミールに関わる光の周波数帯では）。

Boyer の予測は理論的に正しいと証明されている（Kenneth et al. 2002, PRL）。しかし実験で確認されたことは一度もない。
\end{honestbox}

\section{「カシミール斥力」を名乗る実験たち}

\begin{honestbox}[なぜ既存の実験はBoyerの斥力ではないか]
\textbf{Munday et al. (2009, Nature)}: 金とシリカの間にブロモベンゼン液を満たし、カシミール斥力を測定。しかしこれは「液体の誘電率が中間的だから」斥力が生じる（DLP条件）。板の間の\textbf{モード構造は変わっていない}。全モードが存在したまま。素数ミュートではない。

\textbf{Tang et al. (2017, Nature Photonics)}: T字型シリコンナノ構造をスライドさせ、力のベクトル和が正になる配置を発見。しかし\textbf{全ペアのカシミール力は引力のまま}。複数の引力の合計が偶然正の方向を向いただけ。モード構造は変わっていない。

\textbf{Boyer 球殻 (1968)}: 導体球殻のカシミール自己エネルギーが正（外向き圧力）。しかし球のモードは全スペクトル（球面調和関数の全次数）。偶数を消していない。

\textbf{メタマテリアル提案 (Rosa 2008, Leonhardt 2007)}: メタマテリアルでPMCを近似する理論提案。しかしKramers-Kronig制約により、狭帯域の磁気応答は虚数周波数で消える。実現していない。

\textbf{全てのケースで共通}: モード和は $\sum n^{-s}$（全モード）のまま。偶数モードが消えていない。オイラー積は無傷。

\textbf{モード和が $1^3+3^3+5^3+\cdots = \zeta_{\neg 2}(-3)$ に切断された実験は、人類史上ゼロ。}
\end{honestbox}

\section{オイラー積}

\begin{mathbox}[ゼータ関数の「素因数分解」]
リーマンゼータ関数は素数で分解できる（オイラー、1737年）:
$$\zeta(s) = \frac{1}{1-2^{-s}} \times \frac{1}{1-3^{-s}} \times \frac{1}{1-5^{-s}} \times \cdots$$

各因子が1つの素数に対応。

\section{素数2を「消す」}

\begin{mathbox}[素数ミュート]
$p=2$ の因子を除去する:
$$\zeta_{\neg 2}(s) = \zeta(s) \times (1 - 2^{-s}) = \sum_{n \text{ odd}} n^{-s}$$

$2$ の因子を消すと、$2$ で割れる全ての整数（偶数）が和から消え、奇数だけが残る。

3Dでは:
$$\zeta(-3) = +\frac{1}{120} \;\xrightarrow{\text{素数2ミュート}}\; \zeta_{\neg 2}(-3) = -\frac{7}{120}$$

Boyer の計算はまさにこれ: 「偶数モードが消えて奇数モードだけ残る」= オイラー積の $p=2$ 因子の除去。

\textbf{Boyer は「素数2をミュートした」ことに気づいていなかった。}
\end{mathbox}

PMC は要らない。片端を開けるだけ。
\end{resultbox}

\section{巻き付き数——輪ゴムの話}

\begin{storybox}[指に巻いた輪ゴム]
指に輪ゴムを巻くことを考える。

\textbf{0回巻き}: 輪ゴムは指の横にあるだけ。引っ張れば外せる。

\textbf{1回巻き}: 輪ゴムが指を1周している。引っ張っても外せない。切らない限り。

\textbf{2回巻き}: 2周。もっと外せない。

「何回巻いているか」= 巻き付き数。これは0, 1, 2, 3, ... の整数値を取る。

核心: 0回と1回の間には「連続的にいじっても越えられない壁」がある。輪ゴムを伸ばしたり縮めたりしても、巻き数は変わらない。切らないと変えられない。これが「位相的に保護されている」ということ。
\end{storybox}

\section{DD と DN は「巻き方」が違う}

\begin{mathbox}[DD = 巻いていない、DN = 巻いている]
DD（両端閉じ、$\lambda/2$）のモード: 1, 2, 3, 4, 5, ...（全整数）

全整数の世界の「形」を数学で記述すると:
$$K_1(\mathbb{Z}) = \mathbb{Z}/2$$

$\mathbb{Z}/2$ = 「0 か 1 の2択」。巻き付き数がない。
輪ゴムが指の横にあるだけの状態。

DN（片端開放、$\lambda/4$）のモード: 1, 3, 5, 7, 9, ...（奇数のみ）

奇数だけの世界の「形」:
$$K_1(\mathbb{Z}[1/2]) = \mathbb{Z}/2 \times \mathbb{Z}$$

$\mathbb{Z}$ = 「..., $-2, -1, 0, 1, 2$, ... の整数」。巻き付き数がある！
輪ゴムが指を巻いている状態。

\textbf{DD → DN に変えると、巻き付き数が「なし」から「あり」に変わる。}
\end{mathbox}

\begin{storybox}[これが意味すること]
DD の真空エネルギーは負（引力）。巻き付き数がないので、パラメータを連続的にいじれば符号を変えられる。保護がない。

DN の真空エネルギーは正（斥力）。巻き付き数があるので、パラメータを連続的にいじっても符号は変わらない。\textbf{保護されている}。

もしこの保護が物理的に本物なら:
板を離していっても DN の斥力の「正の符号」は保護される。
つまりエネルギー密度は $1/a^4$ で消えていくのではなく、
あるところでプラトーに達する（= 体積効果）。

一方、DD は保護がないので $1/a^4$ のまま消える（= 表面効果のまま）。
\end{storybox}

しかし:
\begin{itemize}
\item QED が実験で確認されたのは DD（巻き付き数なし）だけ
\item DN（巻き付き数あり）で QED をテストした人は\textbf{ゼロ}
\item 「テスト済みの領域で正しい」$\neq$「未テストの領域でも正しい」

前例:
\begin{itemize}
\item ニュートン力学は低速で完璧 → 高速で破れた（特殊相対論）

\item 両方とも「テスト済み領域では正しいが、未テスト領域で新物理が出た」

QED は DD で正しい。DN でも正しいかどうかは\textbf{実験で確かめるしかない}。

\textbf{この実験は「Boyer の予測の検証」であると同時に、
「QED を位相的に異なる真空セクターで初めてテストする」実験。}
\end{honestbox}

\section{繰り込み——無限大を捨てる技法}

\begin{storybox}[QED の秘密]
量子電磁力学（QED）は物理学史上最も精密な理論。電子の磁気モーメントを小数点以下12桁まで正しく予測する。

しかし QED の計算過程では「無限大」が頻繁に出現する。真空エネルギーの計算:
$$1^3 + 2^3 + 3^3 + 4^3 + \cdots = \infty$$

QED のやり方: この無限大を「繰り込み（renormalization）」で差し引く。
最終的な予測（カシミール力など）は、2つの無限大の\textbf{差}として求まり、有限で実験と合う。

\textbf{問題}: 差し引かれた無限大の中身は「ゴミ箱行き」。物理的意味がないとされてきた。
\end{storybox}

\section{ゴミ箱の中身を覗く}

\begin{resultbox}[ゼータ関数による正則化はゴミ箱の中身を見る方法]
$\zeta$ 正則化は「捨てるべき無限大」に有限の値を与える:
$$1^3 + 2^3 + 3^3 + 4^3 + \cdots = \zeta(-3) = \frac{1}{120}$$

QED はこの $1/120$ を「途中計算の数字」として使い、最終的に差を取って捨てる。

しかし $1/120 = \zeta(-3)$ には\textbf{内部構造}がある。オイラー積:
$$\zeta(-3) = \prod_{p\text{ prime}} \frac{1}{1-p^3}$$

素数 $p = 2, 3, 5, 7, \ldots$ ごとの因子に分解できる。繰り込みはこの値を丸ごと捨てるが、\textbf{素数ごとの分解が物理的に意味を持つかどうか}は一度も問われたことがない。
\end{resultbox}

\section{DN 実験 = ゴミ箱を開ける実験}

\begin{resultbox}[何が見つかるか]
DN 配置（$\lambda/4$）は $p=2$ の因子を除去する: $\zeta(-3) \to \zeta_{\neg 2}(-3)$。

これは「ゴミ箱に入っていた $\zeta(-3)$ の中身をいじる」操作。

\textbf{標準QEDの立場}: ゴミ箱の中身は物理に無関係。いじっても予測 $(\alpha = 5)$ は変わらない。

\textbf{我々の予想}: ゴミ箱の中身には素数構造があり、それは物理的。いじると $\alpha$ が変わる。

\textbf{もし $\alpha \neq 5$ が見つかったら}: 繰り込みで捨てていた真空エネルギーの算術的構造が\textbf{ずっと物理的だった}ことになる。QED の予測は正しかったが、何を捨てているかの\textbf{解釈}が不完全だった。
\end{resultbox}

\begin{storybox}[たとえ話: 家計簿]
あなたの月収が 100 万円で支出が 99 万円。手残り 1 万円。

会計士（= QED）は「手残り 1 万円」だけを報告する。月収と支出の内訳は「関係ない」と言う。

しかしもし月収の内訳（給料、ボーナス、副業、投資...）に構造があり、特定の収入源を消すと手残りの\textbf{符号が変わる}（赤字になる）としたら？

QED は「手残りの差」しか見ていない。我々は「収入の内訳（= オイラー積の素数因子）」に意味があると主張し、それを実験で確かめようとしている。
\end{storybox}

\section{なぜ3つの道が必要か}

$\lambda/4$ 共振器は cm サイズで簡単に作れるが、カシミール力は nm サイズでしか測れない。このスケールの不整合を3つの方法で回避する。

%────────────────────────────────────────────────
\section{Route A: ナノスケール DN 構造（室温 AFM）}
%────────────────────────────────────────────────

\begin{storybox}[何をするか（一言）]
ナノスケールで「片端が開いた構造」を作り、金の平板との間に\textbf{斥力}が生じるかを AFM で測る。
\end{storybox}

\subsection{AFM（原子間力顕微鏡）とは}
\begin{storybox}[AFM の仕組み]
極めて細い針（カンチレバー）を表面に近づけて、針にかかる力を検出する装置。

力 → カンチレバーが撓む → レーザーで撓みを読む → 力の大きさと方向がわかる。

感度: ピコニュートン（$10^{-12}$ N）レベル。原子1個にかかる力を測れる。

大学の物理・材料系の学科にほぼ必ずある。1時間 \textyen 5,000〜10,000 で使える（共同利用）。
\end{storybox}

\subsection{DN 表面の候補}
\begin{expbox}[ノイマン条件をナノで作る]
「片端開放」をナノスケールで実現する候補:

\textbf{候補1: 穴あき金属膜}
\begin{itemize}
\item 金の薄膜（50〜100 nm厚）にナノサイズの穴を周期的に開ける
\item 穴の部分 = 電場の境界条件が「開放」に近い
\item FIB（集束イオンビーム）or EBL（電子ビームリソグラフィ）で加工
\item 大学のクリーンルームで製作可能
\end{itemize}

\textbf{候補2: フォトニック結晶表面}
\begin{itemize}
\item 周期構造により特定の周波数帯で「高インピーダンス表面」を実現
\item 高インピーダンス ≈ PMC に近い振る舞い
\item Si のエッチングで製作可能
\end{itemize}
\end{expbox}

\subsection{実験手順}
\begin{expbox}[ステップバイステップ]
\begin{enumerate}
\item \textbf{DD基準データの取得}（1日目）
  \begin{itemize}
  \item 金コートされた2枚の平板（Si基板に金蒸着）
  \item AFM で片方の板を近づけながら力を記録
  \item 距離 $a = 50$ nm 〜 $1\,\mu$m で $F(a)$ を測定
  \item → 引力（負の力）のカーブが得られる（標準カシミール）
  \end{itemize}

\item \textbf{DN構造の製作}（1〜2週間）
  \begin{itemize}
  \item Si 基板に金を蒸着
  \item FIB で周期的なスロット（穴）を加工
  \item スロットのピッチ: 200〜500 nm、幅: 50〜100 nm
  \end{itemize}

\item \textbf{DN測定}（2日目）
  \begin{itemize}
  \item 金平板（DD）の代わりに DN 構造を AFM にセット
  \item 同じ距離範囲で $F(a)$ を測定
  \item → 力が正（斥力）か負（引力）かを判定
  \end{itemize}

\item \textbf{スケーリング解析}
  \begin{itemize}
  \item $F(a)$ を log-log プロットし、傾き $\alpha$ をフィット
  \item DD: $\alpha = 5$（基準、既知）
  \item DN: $\alpha = 5$（標準QED）か $\alpha \neq 5$（新物理）か
  \end{itemize}
\end{enumerate}
\end{expbox}

\textbf{技術的}: FIB でスロットを開けただけで本当に DN 条件が実現するかは不明。穴の形状・サイズ・周期の最適化が必要。初回で成功する保証はない。複数の設計を試す必要がある → コストが倍増する可能性。

\textbf{政治的}: FIB と AFM の共同利用枠を確保する必要がある。大学との関係構築が必須。
\end{honestbox}

\section{Route B: circuit QED（$\lambda/4$ vs $\lambda/2$）}
%────────────────────────────────────────────────

\begin{storybox}[何をするか（一言）]
2本の超伝導共振器（$\lambda/4$ と $\lambda/2$）に量子ビットを入れて、真空揺らぎのシフトが異なることを測定する。
\end{storybox}

\subsection{必要なものの説明}

\begin{storybox}[超伝導共振器]
アルミの同軸ケーブルを $-273$\textdegree C 近くに冷やすと、電気抵抗がゼロになる（超伝導）。
マイクロ波（GHz帯の電磁波）がケーブルの中でほとんどロスなく反射を繰り返す = 共振器。

$\lambda/2$: 両端をショート → 全倍音（$f, 2f, 3f, 4f, \ldots$）\\
$\lambda/4$: 片端ショート、片端オープン → 奇数倍音のみ（$f, 3f, 5f, \ldots$）

同軸ケーブルを切るだけで作れる。長さ $\sim$3 cm で基本周波数 $\sim$5 GHz。
\end{storybox}

\begin{storybox}[トランスモン量子ビット]
超伝導のジョセフソン接合に大きなコンデンサを付けた「人工原子」。

5〜10 GHz で共鳴し、キャビティの真空揺らぎと相互作用する。
真空揺らぎがあるとトランスモンの周波数が微妙にずれる = Lamb シフト。

Google や IBM の量子コンピュータに使われているのと同じデバイス。

\textbf{個人では製作困難}。共同研究先の研究室から借りる or 提供してもらう。
\end{storybox}

\begin{storybox}[希釈冷凍機]
$\sim$10 mK（$-273.14$\textdegree C）まで冷やす装置。

$^3$He と $^4$He の混合液が2相に分離するときの冷却効果を利用。
大学の低温物理学研究室にある（1台 2000万〜5000万円。借りる）。
\end{storybox}

\subsection{実験手順}
\begin{expbox}[ステップバイステップ]
\begin{enumerate}
\item \textbf{共振器の製作}（個人でできる）
  \begin{itemize}
  \item アルミの同軸ケーブル（直径 $\sim$3 mm）を 2 本用意
  \item ケーブル A: 両端に SMA コネクタをハンダ付け → $\lambda/2$
  \item ケーブル B: 片端 SMA、もう片端をカット（オープン）→ $\lambda/4$
  \item 長さを揃えて基本周波数を合わせる
  \end{itemize}

\item \textbf{室温での特性確認}（個人でできる、nanoVNA \textyen 3,000）
  \begin{itemize}
  \item $\lambda/2$: ピークが $f, 2f, 3f, \ldots$ の等間隔
  \item $\lambda/4$: ピークが $f, 3f, 5f, \ldots$ の奇数のみ
  \item 偶奇分離を目視確認
  \end{itemize}

\item \textbf{共同研究先に持ち込み}
  \begin{itemize}
  \item 「この2本の共振器にトランスモンを入れて Lamb シフトを比較したい」
  \item 研究室がトランスモンと冷凍機を提供
  \end{itemize}

\item \textbf{冷却と測定}（研究室にて）
  \begin{itemize}
  \item $\lambda/2$ + トランスモン → 冷却 → 周波数 $f_q^{(\lambda/2)}$ を測定
  \item $\lambda/4$ + トランスモン → 冷却 → 周波数 $f_q^{(\lambda/4)}$ を測定
  \item 差 $\Delta f = f_q^{(\lambda/4)} - f_q^{(\lambda/2)}$ = 偶数モードの真空寄与
  \end{itemize}

\item \textbf{比較}
  \begin{itemize}
  \item $\Delta f$ が標準 QED の計算値（$\sum_{n\text{ even}} g_n^2/(f_q - f_n)$）と合うか
  \item 合えば → QED は DN でも正しい
  \item 合わなければ → DN 領域で新物理
  \end{itemize}
\end{enumerate}
\end{expbox}

\section{Route C: Si-MEMS（Tang 型プラットフォーム）}
%────────────────────────────────────────────────

\begin{storybox}[何をするか（一言）]
シリコン半導体プロセスで DN 型のナノ構造を作り、力を板間隔 $a$ の関数として測定して、スケーリング指数 $\alpha$ を決定する。
\end{storybox}

\subsection{Tang et al. (2017) の装置}
\begin{storybox}[既存プラットフォーム]
Tang のチームが Nature Photonics で使った装置:

\textbf{コムアクチュエータ}: 櫛歯状の電極に電圧をかけて、ナノ構造の位置を精密に制御。1 nm 以下の精度で距離を変えられる。

\textbf{振動ビーム力センサー}: シリコンの微小な梁が振動。力がかかると振動周波数が変わる。周波数の変化から力の大きさを算出。

\textbf{半導体プロセスで一括製作}: フォトリソグラフィ + エッチングで大量のデバイスを一度に作れる。

全てシリコンチップ上に集積。1チップに数十のデバイスを並べて統計を取れる。
\end{storybox}

\subsection{変更点: T字 → DN 型}
\begin{expbox}[Tang のT字構造を DN 構造に変える]
Tang のT字構造: 幾何学的なベクトル和で「斥力方向」を実現（モード構造は変えない）。

\textbf{我々の DN 構造}: 片端が開放されたナノチャンネル（溝）を周期的に配列。

\begin{verbatim}
Tang (T字):          我々 (DN):
 ┌┴┐  ┌┴┐           ┌──┐  ┌──┐
 │  │  │  │           │  │  │  │
─┴──┴──┴──┴─       ─┴──┘  └──┴─
                      ↑開放端  ↑開放端
\end{verbatim}

開放端（オープン）= ノイマン条件。閉じた端（壁）= ディリクレ条件。
構造の半分が DN 条件を持つ。
\end{expbox}

\subsection{実験手順}
\begin{expbox}[ステップバイステップ]
\begin{enumerate}
\item \textbf{デバイス設計}（個人でできる）
  \begin{itemize}
  \item CAD（KLayout 等の無料ソフト）で DN 型構造のマスクを設計
  \item 溝の幅: 100〜500 nm、深さ: 500 nm〜1 $\mu$m
  \item コムアクチュエータと振動ビームは Tang の設計を参考にする
  \end{itemize}

\item \textbf{デバイス製作}（大学のクリーンルーム or ファウンドリ）
  \begin{itemize}
  \item SOI（Silicon on Insulator）ウエハーを使用
  \item フォトリソグラフィでパターン転写
  \item DRIE（深堀りエッチング）で構造を形成
  \item HF（フッ酸）で犠牲層除去 → 可動構造がリリース
  \end{itemize}

\item \textbf{DD 基準測定}
  \begin{itemize}
  \item 溝のないフラットな面同士で力 vs 距離を測定
  \item $F \propto a^{-5}$（引力）を確認 → 基準データ
  \end{itemize}

\item \textbf{DN 測定}
  \begin{itemize}
  \item DN 構造の面 vs フラットな面で力 vs 距離を測定
  \item 距離 $a$ を 50 nm 〜 数 $\mu$m でスキャン
  \end{itemize}

\item \textbf{スケーリング解析}
  \begin{itemize}
  \item $F(a)$ を log-log プロットし、傾き $\alpha$ をフィット
  \item DD と DN を同一チップ上で比較（制御実験）
  \item DD: $\alpha = 5$（確認）、DN: $\alpha = ?$
  \end{itemize}
\end{enumerate}
\end{expbox}
